Question-1: 
Write a python program for blinking LED’s and Traffic light
Solution:
BLINKING LED’S
from gpiozero import LED
red = LED(22)
amber = LED(27)
green = LED (17)
red.blink(1, 1)
amber.blink(2, 2)
green.blink(3, 3)
TRAFFIC LIGHT
 from gpiozero import LED
 from time import sleep
 red = LED(22)
 amber = LED(27)
 green = LED(17)
 while True:
 
red.on()
 
amber.off()
 
green.off()
 
sleep(1)
 
red.off()
 
amber.on()
 
green.off()
 
sleep(1)
 
red.off()
 
amber.off()
 
green.on()
